Health Risk Checker Using Python
================================

Project Description
-------------------
This project is a simple healthcare program developed using Python. 
The program checks a person's health risk based on three parameters:

1. Age
2. Blood Pressure
3. Heart Rate

Based on these inputs, the program analyzes the data and determines whether 
the person may have a high health risk.

Features
--------
- Takes user input for health parameters
- Uses conditional statements to analyze health risk
- Displays appropriate health messages
- Simple and beginner-friendly Python project

Technologies Used
-----------------
- Python
- Conditional Statements (if, elif, else)
- User Input Handling

How to Run the Program
----------------------
1. Open the project folder.
2. Open the file "health_checker.py".
3. Run the program using Python.
4. Enter age, blood pressure, and heart rate when prompted.

Example Output
--------------

Enter your age: 55
Enter your blood pressure: 150
Enter your heart rate: 90

High Health Risk. Please consult a doctor.
